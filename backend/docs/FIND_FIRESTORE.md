- Click it if it's a dropdown
- This might open navigation menu

### Step 3: Check Left Edge of Screen
- Sidebar might be collapsed (just showing icons)

### Step 5: Direct URL
Try going directly to:
- https://console.firebase.google.com/project/YOUR-PROJECT-ID/firestore

Replace YOUR-PROJECT-ID with your project ID.
